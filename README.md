## Descripción del taller
Este repositorio contiene la implementación y resolución de dos problemas clásicos de lógica, Sudoku y Kakuro, modelados como problemas de Satisfacción por Restricciones (CSP) utilizando el lenguaje de programación MiniZinc. El objetivo es aplicar técnicas de programación para resolver estos problemas aplicando restricciones.
